🆔 项目名称：Tinymind

⭐️ 项目功能：使用 Github 写博客

📁 项目简介：一个开源的通过 GitHub 轻松撰写和同步博客及个人想法的工具。

只需使用 GitHub 账号登录，Tinymind 会自动创建一个名为“tinymind-blog”的公共仓库，所有在 Tinymind 上撰写的内容都会即时同步到该仓库中。

不需要服务器，所有数据直接存储在用户的 GitHub 账户中，支持 Markdown 格式。

🗣Topit：macOS 上的应用窗口置顶工具

📁 软件简介：Topit 是一个开源工具，专为 macOS 用户设计，可以让用户将任意多个应用窗口强制置顶显示。用户可以自由移动、缩放或与被置顶窗口进行交互，提高工作效率。使用 Topit，只需启动应用，选择需要置顶的窗口，然后点击“立即置顶”即可。
📁 特色：
应用窗口置顶：将任意多个窗口置顶显示。
用户交互：允许用户与置顶窗口进行自由移动、缩放等交互。
简单易用：启动 Topit，选择窗口，点击“立即置顶”即可。
开源项目：代码开源，用户可以自由使用和贡献代码

🗣Markdown Poster：Markdown 转海报图片，用 Markdown 制作优雅的图文海报

📁 网站简介：Markdown To Poster 是一个开源工具，能够将 Markdown 内容渲染成适合社交媒体分享的图片。它提供了一个内置模板和 9 个自定义主题，支持模板扩展，并允许用户复制输出为图像。此外，该项目还包含了一个内置的 WEB 编辑器，可以一键部署，方便用户在线编辑 Markdown 并快速生成海报图片。

同时网站还提供一个免费的海报渲染 API，可以集成到你的网站、Bot、ChatGPT、Coze 等平台。

📁 特色：
Markdown 渲染：将 Markdown 内容转换为视觉吸引人的社交媒体图片。
内置模板：提供一个基础模板，并支持用户自定义模板。
自定义主题：内置 9 个主题，用户可以根据需要选择或创建新主题。
一键复制为图像：方便用户将生成的海报直接复制为图像使用。
内置 WEB 编辑器：提供在线编辑器，支持一键部署，方便在线编辑和转换 Markdown。
